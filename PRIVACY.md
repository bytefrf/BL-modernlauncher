# Privacy Policy — BL-modern TFGM Launcher

_Last updated: 2026-06-16_

The BL-modern TFGM Launcher is a Windows launcher for the TerraFirmaGreg-Modern
Minecraft modpack. This document explains what data the launcher collects and how
it is used.

## What we collect (anonymous telemetry)

When telemetry is enabled, the launcher sends **anonymous** technical events to our
server (`https://bl-modern.ru/api/`). These events contain:

- A random, locally generated client identifier (GUID) — not linked to any account
  or personal identity.
- Launcher and modpack version.
- Operating system version and architecture, CPU core count, total/available RAM,
  system locale, .NET runtime version, screen resolution.
- A short non-reversible hash of the machine name (used only to count unique machines).
- Application events: launch started/succeeded/failed, install completed, game session
  ended or crashed, and related diagnostic fields (e.g. exit code, crash category).

We **do not** collect: your Minecraft account credentials, real name, email address,
chat messages, file contents, or any personally identifying information.

## How the data is used

The data is used solely to monitor launcher stability, diagnose crashes and
installation problems, and improve the launcher and the game server. It is not sold
or shared with third parties.

## Opt-out

Telemetry can be disabled at any time in the launcher's **Settings** screen
("Отправлять анонимную статистику"). When disabled, no events are sent.

## Support logs

If you use the in-launcher "send support log" feature, the launcher uploads diagnostic
logs (game logs, system info) to help us assist you. This is initiated only by your
explicit action.

## Code signing

Windows builds of this launcher are code-signed using a certificate provided by the
[SignPath Foundation](https://signpath.org/) (free code signing for open-source projects).
